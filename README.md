# 玉米预测时序项目

本仓库按“算子 + 材料”的方式组织玉米价格预测相关资产。

- `玉米预测/operator/model/`: 模型算子本体，包括模型结构、模型内部层、工具和单模型配置。
- `玉米预测/datasets/`: 被模型消费的数据材料，包括 raw、processed、metadata 和 factors。
- `玉米预测/scripts/`: 运行、同步、聚合和维护脚本，不属于模型算子本体。
- `src/`: 训练、回测、数据处理、评估流程代码，以及旧 import 路径的兼容入口。
- `layers/`、`utils/`: 指向新模型层的兼容入口。

`operator` 不包含评估、预测、调度和报告脚本；这些入口统一放在 `玉米预测/scripts/`。
